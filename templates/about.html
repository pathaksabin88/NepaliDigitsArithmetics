Project Title

Arithmetic Operation on Handwritten Nepali Digits Recognized Using Neural Network

Abstract

Characters recognition plays an important role in the modern world. It can solve more complex problems and make humansâ€™ job easier. An example is handwritten character recognition. This project is all about the recognition of handwritten Nepali digits and perform arithmetic operation (Division, Multiplication, Addition, and Subtraction) as specified in the given handwritten Nepali digits string. The output of the expression is printed step by step in an image so that the user can compare their result to that of image. This project can motivate the children to learn arithmetic operation as they feel really excited getting the problems solved in digitized way.
